About:
Use this script to rename CKL files based the contents of the CKL file.

Instructions:
1. Put your old CKL files in the folder called "ckl"

2. open "run.ps1" in PowerShell ISE

3. Configure (see note below)

3. Your new files will be copied to the folder called "updated ckl files" 

*Notes*
○ The script can be configured to:
	• use different delimiters to separate the details.
	• Set system name
	• Use a different structure, scroll to the bottom and 
	  rearrange the variables however you want

○ See code comments in green text for more details
